Release Notes - 2.1.4¶

NAO¶

Basic Channel - improved¶

The Basic Channel content has been reworked and improved, especially the content of the conversation for English, French, and Japanese languages.

For further details, see: Basic Channel - What can I say to NAO.

Choregraphe¶

Supported Languages - better coverage¶

The level of support and integration of the following languages has been improved, they are now all available in Dialog boxes:

  • Arabic
  • Czech
  • Danish
  • Dutch
  • Finnish
  • Greek
  • Norwegian
  • Polish
  • Brazilian
  • Portuguese
  • Russian
  • Swedish
  • Turkish

For further details, see: Available languages.

NAOqi¶

ALVisionRecognition - improved¶

Several issues, impacting performances have been solved.

SDKs¶

Java SDK available¶

The Java SDK provides Java bindings to:

  • call remote services,
  • create new services, and
  • react to events.

For further details, see: Programming.
