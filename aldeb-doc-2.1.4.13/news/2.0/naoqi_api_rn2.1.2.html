NAOqi API & SDks 2.1.2¶

ALAutonomousMoves: New methods¶

New methods to define a background strategy when the Aldebaran robot is not busy with motion instructions:

  • ALAutonomousMovesProxy::setBackgroundStrategy()
  • ALAutonomousMovesProxy::getBackgroundStrategy()

For further details, see: Background Strategy.

ALRobotPosture: algorithm improved¶

The algorithm trying to go from lying to rising posture has slightly evolved. This fixes several issues, in particular when NAO is lying on his side.

For further details see: ALRobotPosture and NAO has fallen.

ALTextToSpeech: tts technology has changed¶

ALTextToSpeech is based on speech synthesizers - or speech engines.

According to the selected language, a specific engine is used.

The following language package, formerly using ACAPELA, are now based on Nuance TTS Technology:

‘Chinese’, ‘English’, ‘French’

For further details see: ALTextToSpeech.

PackageManager: New methods and signals¶

  • PackageManager::packages2()
  • PackageManager::package2()
  • PackageManager::onPackageInstalled2
  • PackageManager::onPackageRemoved2
