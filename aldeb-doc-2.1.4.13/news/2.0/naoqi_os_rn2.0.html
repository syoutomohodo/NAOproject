NAOqi OS 2.1¶

OpenNAO becomes NAOqi OS¶

Text editor¶

NAOqi OS is provided with nano.

Note that the text editors previously provided, qemacs and vim, are no longer available.

NAOqi OS VM¶

Because the main use of the NAOqi OS VM is to build open-source dependencies not provided by the Aldebaran’s SDKs, the NAOqi executable has been removed from it.

Miscellaneous¶

The embedded NAOqi OS has evolved; among the noticeable changes, there are:

  • Switch web server from lighttpd to nginx;
  • Audio modules are now using pulseaudio on the robot.
