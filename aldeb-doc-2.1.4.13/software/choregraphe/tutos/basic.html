Playing music¶

Step Action

Drag and drop the Audio > Sound > Play Sound box onto the Flow Diagram panel.
Connect its input and output to the diagram input and output.
Click on its parameter button.
Click the Browse button then click Import file button to import an audio file then select it.

Click OK button twice.

The file is stored. You will not have to upload it again.

Click the Play button to launch your behavior.

The music file you have selected is played on NAO.
