Behavior¶

What is a Behavior¶

Creating a Behavior is an easy way to make the robot talk, walk, dance, play a sound, etc.

A Behavior contains Boxes which are sequentially or simultaneously executed.

Inside the Project folder, the Behavior is stored in a .XAR file located in a sub-folder.
